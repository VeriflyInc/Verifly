FEATURE WHY VERIFLY PURPOSE

VERIFLY

AI時代の音楽に、人間が作ったことの証跡を残す。 Veriflyは、作曲者・演奏者・共同制作者・活動履歴をもとに、 楽曲へ信頼のデジタルラベルを付与する認証レイヤーです。

認証を行う
Creator proof profile / history
Collaboration proof mutual approval
Work proof project files / stems
Activity proof live / release archive

FEATURE

Veriflyは、AI楽曲を一方的に検出するサービスではありません。 人間の制作活動に紐づく証拠を複数のルートから集め、 楽曲やアーティストに認証ラベルを付与します。

活動履歴

ライブ、リリース、SNS投稿、過去作品、メディア掲載などを確認します。

共同認証

バンドメンバー、プロデューサー、エンジニアなどの相互認証を扱います。

制作証跡

DAWプロジェクト、ステム、録音過程、制作メモなどを確認します。

WHY VERIFLY

AI生成音楽が自然に流通するほど、リスナーは 「誰が、どのように作った音楽なのか」を見分けにくくなります。 だからこそ、プラットフォーム上に透明性を示すラベルが必要になります。

VeriflyはAIを否定するためではなく、人間の創作物を 人間の創作物として扱うための仕組みです。

PURPOSE

Veriflyは、Spotify、YouTube、Discordのような既存プラットフォームに 組み込める音楽真正性のインフラを目指します。 新しいSNSを作るのではなく、すでに音楽が消費されている場所へ 信頼のレイヤーを提供します。
